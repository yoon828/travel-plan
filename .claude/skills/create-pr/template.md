각 섹션의 주석(`<!-- -->`)을 참고해서 내용을 채우고, 주석은 최종 결과물에서 제거한다.

  이번 PR에서 여행 생성 폼에 멤버 등록 기능을 통합해 한 번에 처리할 수 있도록 개선했습니다.
